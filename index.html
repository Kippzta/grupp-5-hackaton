Reflektion kring grupparbete med Git Flow

Under vårt grupparbete har vi arbetat med Git Flow som arbetssätt och fått en djupare förståelse för hur det fungerar i praktiken. Samarbetet i gruppen fungerade väldigt bra, och vi kompletterade varandra på ett effektivt sätt. Omar hade sedan tidigare erfarenhet av Git Flow, vilket var till stor hjälp för både Erik och Jonatan som kunde lära sig mycket och snabbt komma in i arbetssättet. Det gjorde att hela gruppen fick en bättre förståelse för hur man strukturerar sitt arbete med versionhantering, feature branches, pull requests och merges på ett effektivt sätt. Vi upplevde att det var tydligt vem som gjorde vad, och att kommunikationen flöt på bra genom hela processen. Att jobba enligt Git Flow bidrog också till att vi undvek konflikter i koden och höll god ordning på våra ändringar. Sammanfattningsvis var det en lärorik uppgift där vi både utvecklades tekniskt och som grupp.
